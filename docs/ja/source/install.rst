.. _install:

============
インストール
============

インストールは `easy_install` や `pip` コマンドで行えます。

easy_install によるインストール
===============================

::

  $ sudo easy_install bpmappers

pip によるインストール
======================

::

  $ pip install bpmappers

PyPI上のページ
==============

ソースコードはPyPI上のページからも入手できます。

:URL: http://pypi.python.org/pypi/bpmappers
